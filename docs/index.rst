Welcome to Donphan
==================

Donphan is an easy to use asynchronous `Postgresql`_ database wrapper for python 3.9+.

.. _Postgresql: https://www.postgresql.org/

**Features:**

 - Object oriented Table design using type-hinting.
 - Simple to use helper methods for ``INSERT``, ``FETCH``, ``UPDATE``, and ``DELETE`` database queries.
 - Support for Custom SQL types.
 - And much much more
